# Bulletlane: v13
- Added back ads.
- Added a bullet damage increase for using the laser car.
- Added a menu to buy coins using gems.
- Added 8 new tiers of enemies.
- Increased the max speed of the game from x2.5 to x3.5.
- Changed the sound effects for getting a powerup and healing.
- Added a new car called the Blast Car, it costs 750 Gems and fires a large AOE blast every so often.
- Fixed the UI a bit.
- Tweak the laser car so it fires regular bullets & has a beam still.
- Fixed tint/texture not working on game over.
- Updated the engine to 4.5.1.

Now out on [Google Play](https://play.google.com/store/apps/details?id=cc.minota.road) and the [App Store](https://apps.apple.com/us/app/bulletlane/id6743356289).
